# OpenAI API Adapter DeepBill 

## 📋 Обзор проекта

**DeepBill** — рабочий адаптер, который позволяет использовать веб-чат DeepSeek через OpenAI API-совместимый интерфейс.

Проект подходит для интеграции с мультиагентными системами, CLI-инструментами, локальными фермами, кастомными пайплайнами и другими решениями, которые уже работают с OpenAI-compatible API.

## 🚀 Возможности

* OpenAI API-совместимый интерфейс
* Рассуждения, поиск в интернете
* Работа с tools / function calling
* Поддержка системного промпта
* Удобный GUI
* Счетчик токенов (по кол-ву текста) для вашего CLI агента.
* Контекст 128к проверен, 256к и более тестируюится.
* Авто-contune на странице дипсик.

## 🛠 В разработке

В ближайших обновлениях планируется добавить:

* **Мультиаккаунт** для ускорения обработки запросов
* **Умную оркестрацию чатов** — адаптер сможет предлагать модели переход в другой чат через tool-вызов

## ⚠️ Важно

DeepBill работает до тех пор, пока доступен используемый аккаунт и веб-интерфейс DeepSeek. Архитектура проекта продумана с учётом устойчивости, расширяемости и дальнейшего развития.
